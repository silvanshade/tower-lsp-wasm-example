## Demo

You can experiment with a live demo of the example server integrated with an in-browser editor here:

https://silvanshade.github.io/tower-lsp-web-demo/

## Building

```sh
cargo install cargo-make
cargo make deps
cargo make build
```

## Running

```sh
cargo make run
```
